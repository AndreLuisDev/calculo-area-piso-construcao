## Calculadora de Área de Piso
Este programa calcula a área de piso e aplica descontos conforme a metragem.


- Esse mini projeto partiu da CURIOSIDADE dos estudos essenciais sobre Paradigmas
da Linguagens de Programação e a compreensão dos ELEMENTOS DA LÓGICA DE PROGRAMAÇÃO;

- Conforme há descrição do arquivo <Exercicio_DescreverElementosLP.txt> foi necessário criar
uma TEXTO para que um desenvolvedor pudesse desenvolver uma aplicação a partir da descrição feita,
apontando quais eram as estruturas necessárias a serem implementadas;

- A ideia era aplicar o conhecimento sobre os termos/elementos básicos da programação como VARIÁVEIS, IDENTIFICADOR,
TIPO DE DADOS, BLOCO DE INSTRUÇÃO, ESTRUTURAS DE CONTROLE DE FLUXO e outras sob uma perspectiva geral.

- A descrição que fiz do Exercicio_DescreverElementosLP não postei ainda, pois a matéria está sob avaliação e evitar plágio.

Depois subo aqui o que descrevi a partir do Exercicio_DescreverElementosLP.



### Como executar
1. Instale Python 3.x
2. Execute com: `python CalculoAreaPiso.py`
